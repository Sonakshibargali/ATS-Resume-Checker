# ATS Resume Checker

An AI-powered ATS (Applicant Tracking System) Resume Checker that helps job seekers evaluate their resumes against job descriptions, identify missing keywords, and improve their chances of passing ATS screening.

## Live Demo

🚀 Live Application:
https://ats-resume-checker-p3bh4vldcckpuou8pwwymp.streamlit.app/

---

## Features

* Upload Resume (PDF)
* Paste Job Description
* ATS Match Score Analysis
* Resume Keyword Extraction
* Missing Keyword Detection
* Resume Improvement Suggestions
* AI-Powered Resume Evaluation
* Skills Analysis
* User-Friendly Dashboard

---

## How It Works

1. Upload your resume in PDF format.
2. Paste the target job description.
3. The system extracts and analyzes resume content.
4. Important skills and keywords are identified.
5. ATS compatibility score is generated.
6. Missing keywords are highlighted.
7. Suggestions are provided to improve the resume.
